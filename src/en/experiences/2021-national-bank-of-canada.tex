    \chapter{Août 2021/Juillet 2022 Banque Nationale - Architecte Logiciels (Consultant)}
        Conception d'une architecture de micro-services dans AWS EKS pour synchroniser une application de base de donnée
        sur site avec les applications de la banque dans le cadre du projet Services numériques aux entreprises.
        Création d'un dépôt GitHub pour centraliser les fichiers OpenAPI afin de forcer le développement des contrats
        avant le code.
        \section{Banque Nationale}
            \begin{description}
                \item[Lieu] Montréal, Canada.
                \item[Nombre d'employés] 3 400 employés.
                \item[Activité] Services financiers.
            \end{description}
            \subsection{Services numériques aux entreprises (BNE)}
                \begin{itemize}
                    \item Création de l'architecture logicielle
                    \item Maintenance de la documentation
                    \item Présentation de l'architecture aux architectes applicatifs
                    \item Support des développeurs
                \end{itemize}
            \subsection{Centralisation des fichier OpenAPI}
                \begin{itemize}
                    \item Création du dépôt dans GitHub
                    \item Création et maintenance de la documentation
                    \item Présentation de la solution aux architectes applicatifs et aux architectes de solutions
                    \item Support des développeurs pour intégrer la génération du code à partir des fichier OpenAPI
                \end{itemize}
            \subsection{L'environnement de travail}
                \begin{itemize}
                    \item Environnement SAFe (équipe applicative de plus de 100 collaborateurs)
                    \item AWS
                    \item ArchiMate®
                    \item EKS
                    \item SQS
                    \item Kubernetes
                    \item Kafka
                    \item Docker
                    \item Java 11
                    \item Spring webflux
                \end{itemize}
